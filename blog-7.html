Converting any HTML template into a Django template

  • 19 July 2020
  • Sankalpa CM, 1NC17CS080 CSE,NCET
  • Coding

Django is a free and open-source web framework. Like many knowns about the free part, open-source part, and you may think what is a framework. Now framework is a combination of certain components and packages. Ofcource if you want to build website there are certain things you need as readymade for example, if you want build any web application there is no need to build it by from the scratch, so in that case we can use framework which reduces our time consumption. But Django only not the single framework available, we have many frameworks available, and Django we can say as one of the famous web application framework.

To build a website we have the HTML, CSS, JS. Where HTML is used to design the page you can think where HTML as make for your website, and then we have CSS if you want to improve your design and if you want to have that unformatted in design there you will use CSS, and JS used to make your website to interactive. These all we use as frontend, for backend and to make website dynamic we have like servlet, JS, PHP, ASP and python.

To use python as a backend , we use Django. So Django can be used as a dynamic backend for the website.

Any HTML, CSS, JS or BootStrap template can be converted into a Django compatible template. The trick usually is to add {% static %} tags for the js, css and image files that we want to include in the html files. That can become a very lengthy task to do for a complete page.

So now let’s get start for setup so if you want to work with Django as we mentioned Django is the web framework for python that means if you want work with Django you need a python setup in your system.

To install python, just go to google and search python download then search for latest version of python.


After installing python open command prompt to check the version of the python. Give this command then you can get the version of python.

python --version
Next thing you need is pip. Pip is the installer manager for python, pip comes as default when you install python package. To check the version of the pip.
pip –version

Now we got python and pip , now let’s install Django. To have the different set for Django not to disturb other project in your system to make isolated space for project which will not affect the other project you have. this can be done by installing the virtual environment wrapper. 
pip install virtualenvwrapper-win
This commend will install the virtual environment. By this we get the environment, but still we need to create the environment
mkvirtualenv test7
here Test7 it is just name of the environment u created. u can give any name to your wish.

you can check there is Test7 just before the command line.

To install Django.
pip install django
To check the version of Django.
django-admin --version
Now we installed all, now let’s create a folder which includes all the files 
mkdir Myprojects
here MyProject is just name of the folder u created. u can give any name to your wish.
To move into that folder.
cd Myprojects
Now to create a project.
django-admin startproject Myprojects
here MyProject is just name of the project that u going to create u can give any name to your wish.
Here I gave same for folder and project u can give same or different name for both as of your wish.

To access and manage the project
dir
To run the project on the server django provides you a server.
python mange.py runserver


Now copy and paste the server Ip in browser

Now open visual studio


1. start
2. open folder

To create the app in visual studio. Open terminal which is above in visual studio. Now in command prompt 
workon test7
After this to start app 
python manage.py startapp Myapp

Here you can see Myprojects is the project name and Myapp is the app name.


Now create a template folder in MYprojects/ and create urls.py file in MyProjects/Myapp. Now in template folder create the file named as index.html in MyProjects/template/.
Now any project which u want to convert it into django you must copy the HTML index file of the project to the visual studio. In in MyProjects/template/index.html/.
Now from visual studio u have to install the Jinja (template engine)
In app which u created urls.py in MyProjects/Myapp/urls.py add the following code: 
from django.urls import path
from.import views
urlpatterns = [path(" ", views.index,name= "index")]

In app, which u created views.py, in that add a view for creating rendering the index page. so modify the MyProjects/Myapp/views.py file and add the following:
from django.shortcuts import render

def index(request)
 return render(request, "index.html")

In Myprojects/urls.py add this path to include app url root add the following:
path('', include("Myapp.urls")),

Then u run the server in terminal
python manage.py runserver
Now we need to create a folder static/ inside the MyProject/ folder and copy all the static files associated with the project there which in this case such as images, js, plugins, styles folder. So let us directly move the folder to the MyProject/static/ directory. The file structure will now look something like this:


Now go to Myprojects/settings.py there u can find at the bottom static files where to add the path for this static files. To work static file we need to do still tow more settings. At 1st u need to specify that where u will file all the files. So by this now we saying django where all static files is. 
STATIC_URL = '/static/'
STATICFILES_DIRS = [ 
     os.path.join(BASE_DIR, 'Myprojects/static')
]
STATIC_ROOT = os.path.join(BASE_DIR, 'assets')
Here we joinig the base part with the Myprojects/static/. Now django helps us to puts all static folder in that as assests. And now u can refer them using static.
Now u needed to create a assests floder to do that in terminal. code as follows:
python manage.py collectstatic
Now u can able to see a assets folder which contain all the static files.
Now we need to change the all urls in Myprojects/template/index.html which all are in the form of href to django understandable language. So just before every path you need do one change.
Example :

link rel="stylesheet" type="text/css" href="styles/bootstrap4/bootstrap.min.css"


U need to change by adding the { % static ‘ ………………….’ %} after the href

As this it follows:

link rel="stylesheet" type="text/css" href="{% static 'styles/bootstrap4/bootstrap.min.css' %}"


Now this says that all the path now it will be coming from the static folder. u need to do same changes for every path.
After this to load that static flies We need to include the line {% load static %} at the top of the html page in order to be able to use the static tag.
{% load static %}
Here we changed everything of project to django .Now run the server where u can able to see the dynamic website which is in django. Now u still feel it as static website only u can ask also what is dynamic here, Now since we are sending all this data from django we can make dynamic by using views.py.
Upto now what we did is :
Basically we are copying a template from any projects of yours that we are saying hey I will be using same template which as some static files but django need to handle that static files so we need to do is that we have to create a folder you can name it anything you want may be static or assest may be as of your wish and these is where u have to paste all the static files and then in the settings.py we specific that from we will get all the static content and how to refer it then in which folder put it so that we got in assets folder here we have all the static files. After this adding path as static in index.html.
