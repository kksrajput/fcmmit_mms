What you will be expecting with this blog?


As the title suggests, It will truly be a guide in making you a script kiddie. This blog will provide you with some of the facets of kali linux tools, since there are many of them, I will be providing you with some of the easy to use and widely popular in tech sector which has an utmost importance in the hackers arena.

You will be a script kiddie, but will have a skillset to break into systems which will define you as a “Hacker”.


What’s a Script Kiddie?


A Script kiddie is someone who uses software written by someone else to break into a computer system.

It might sound demeaning like using someone’s scripts or using someone’s software but it’s also a blatant truth that to write a script first you need to understand how to utilise one.

Getting Started



Requirements (to follow this blog practically):

  • A normal PC of any specs
  • Vmware or Virtual Box and A kali Linux vm
  • A Wlan Adapter( for penetration testing)

    • Note: If you will go for your laptop’s wifi then you have to install kali along with the dual boot loader for ex:grub to continue using your host os

    Why Kali Linux?

    Kali has numerous inbuilt tools for penetration testing so rather installing many of the tools on a different linux distro we can have everything at one place.

    Setup

    Complete Guide on how to Set up Kali on your PC

    Step 1: Download Vmware

  • Download Vmware and install it on your system.You can get the Setup from here

    • VMware Workstation is a virtual machine software that is used for x86 and x86-64 computers to run multiple operating systems over a single physical host computer. Each virtual machine can run a single instance of any operating system (Microsoft, Linux, etc.) simultaneously



    Step 2: Download the Kali Linux VM

  • Download Kali Linux Virtual Machine form this link

    • Select your option according to your system support i.e. x64 bit or x32 bit
    Note: If you will download the kali linux iso rather than its Virtual Machine then you have to install it on vmware which will consume a lot of time and effort.



    Step 3: Open the virtual machine in Vmware

  • Extract the downloaded file.
  • Open Vmware and Press Open.
  • Go to the extracted location in the popup, you will get a file Kali-Linux-2020.2-vmware-amd64.vmx, Select and Open this VM.


    • Step 4: Up and Running

  • Click on Power on the virtual machine.
  • Select Kali/GNU Linux.
  • Both username and password are kali.

  • You will get the home screen of Kali Linux.

  • Fire up the terminal.
  • Type the following command and press enter.
    • sudo su
  • Enter "kali" which is your password and press enter.

    • Once you are done, your terminal will look like this:

    This step is just to ensure the super user previlages of the distro and to check if you have done everything right upto this point.
    If you have any doubt upto this point you can contact me.

    There is one more way to install Kali i.e directly partiioning your disk and using it from there.
    However, I have not discussed it as it will take much more time and effort.

    That's it! You have set kali up and running.
